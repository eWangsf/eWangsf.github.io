Sublime Text3 之 Package Control

Sublime Text没啥好介绍的了，反正大家都是人手一个的，这个文主要想记录一下在使用sublimetext过程中遇到的一些坑，解决方案，好用的sublime text3的插件以及杂杂的一些个人tip，反正很杂就对了。

no-package-availabel

        

  • 看到的时候我整个人是有点懵的，刚开始还以为是网络的原因，直到连续几天想装一直都是这情况，感觉不太对，想到sublime有个控制台就去打开了(顶部view-show console):

    果然是因为连不到，其实连不连得上可以ping一下看看的：ping sublime.wbond.net，我ping的时候丢包100%。
  • 找个可以翻墙的机子，把这个文件下下来：channel_v3.json
  • 把下下来的文件存到某个目录，再在那个目录启动一个服务器，我是放到”Documents”下，再用python启的。
    • cd Documents
    • python -m SimpleHTTPServer 1111 &.
      打开localhost:1111，如果看到文件内容就是对了
  • 修改sublime package中package control的setting:sublime text - preferences - Package settings - Package Control - Settings User - 加上"channels": ["http://127.0.0.1:1111/Documents/channel_v3.json"]

  • 重启sublime生效，然后就可以正常使用package control了

  • 注意：在修改settings以及使用的时候都要保持python服务器启动状态。
