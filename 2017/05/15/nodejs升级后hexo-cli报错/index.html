nodejs升级后hexo-cli报错

出bug了

以前用的node是v6.3.0的，hexo运行的好好，前段时间项目需要就更新了node，结果！刚刚hexo server就报了一堆错：

然后啥都没想一通乱npm rebuild、npm update、npm cache clean，并没什么用
把错误信息丢到google之后出来的issue也是各种无法解决，也有解决方法,不过是nodejs降级，然后本来想放弃的我！瞄了一眼错误信息！
ヾ(｡｀Д´｡) 这！不是hexo-cli的问题吗，本着程序员的基本觉悟（重装试试？
然后

修复步骤

  • npm cache clean
  • npm install hexo-cli -g(装完后hexo -v可以看到hexo-cli升级了)
  • npm rebuild
